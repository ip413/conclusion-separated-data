Blog about data and analysis.

Link: [https://coseda.xyz](https://coseda.xyz)


## Useful commands

Start local jekyll
```bash
rm -rf _site & bundle exec jekyll serve --future --incremental --watch
```

Compile scss
```bash
sass _sass/minima.scss assets/main.css
```

Deployment
```
firebase deploy
```
